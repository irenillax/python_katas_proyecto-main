Proyecto Katas de Python.

Este proyecto recopila la resolución de 41 ejercicios o Katas en Python. El objetivo es fortalecer las bases de la programación estructurada y funcional, orientada a objetos.
Cada kata representa un pequeño reto que desarrolla una habilidad específica del lenguaje: desde estructuras condicionales y bucles, hasta clases y manejo de errores. Los ejercicios fueron resueltos de forma progresiva, comentando los pasos más complejos y aplicando buenas prácticas de código legible y reutilizable.

•	Técnicas utilizadas:
1.	Funciones puras y expresiones lambda.
2.	Uso de las funciones integradas map(), filter() y reduce().
3.	Control de flujo y manejo de excepciones (try/except).
4.	Definición de clases, atributos y métodos.
5.	Entrada y validación de datos por consola.
Se completaron 41 ejercicios, cubriendo desde lógica básica hasta conceptos intermedios de programación orientada a objetos.

•	Resultados clave:
1.	Comprensión práctica de estructuras condicionales y bucles.
2.	Dominio del uso de map(), filter() y reduce() para manipulación funcional de datos.
3.	Implementación de control de errores robusto con excepciones personalizadas.
4.	Creación de clases como Arbol y UsuarioBanco, aplicando principios POO.
5.	Capacidad de convertir problemas reales en soluciones programáticas simples y eficientes.
   
Conclusión: El proyecto refuerza la lógica algorítmica, fomenta la lectura comprensiva del código y sienta una base sólida para el análisis de datos en Python.
